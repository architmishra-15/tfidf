# TF-IDF

This project is the implementation of [tf-idf](https://en.wikipedia.org/wiki/Tf%E2%80%93idf) in C++.

## Dependencies

The only dependency of this project is [`jsoncpp`](https://github.com/open-source-parsers/jsoncpp)
